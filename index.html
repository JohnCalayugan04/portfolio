Smart Biomass Monitoring System

An embedded hardware-software system designed to monitor a copra dryer. It tracks dual-zone temperatures (combustion and drying) and calculates real-time air quality metrics (CO2, SO2, NOx).


Key Features: Custom C++ signal filtering algorithm for noisy analog inputs, dynamic color-coded TFT user interface, and sensor calibration.


Tech Stack: C++, Microcontrollers, SPI Communication, MAX6675 & MQ135 Sensors, LovyanGFX.
